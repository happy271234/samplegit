Git

Git？　

Gitについて簡単なまとめを記載したいと思います。

Gitとは?

ソースコードや変更履歴を管理・統合できる、分散型バージョンシステムです。

Gitのメリットとは？

1, チームでファイルや変更履歴を共有できます。
2, チームでの共同開発を効率化できます。
3, 簡単に古いバージョンに戻せます。

Gitのデメリットとは？

チーム全体でGitについての理解が必要になります。

GitとGitHubの違い

GitとGitHubは、別の意味合いがあります。それぞれの違いとは？

ツールとサービスの違い（メールとGmailの違いのようなもの）

Gitは、ソースコードを保存した履歴（バージョン）を管理するツールです。
GitHubは、クラウド上でGitのバージョンを管理できる、webプラットフォームです。
